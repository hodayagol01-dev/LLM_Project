# PLAN — Assignment 04

## Goal
Create a reproducible agentic workflow for generating a professional academic PDF using LaTeX, LuaLaTeX, a custom CLS file, Skills, and QA checks.

## Work Plan
1. Create planning files: PRD.md, PLAN.md, TODO.md.
2. Create LaTeX files: main.tex, custom CLS file, references.bib.
3. Define agent Skills for content, LaTeX generation, and QA.
4. Compile the PDF with LuaLaTeX.
5. Add QA logs and screenshots.
6. Update README files with setup, execution, output, and limitations.
7. Commit and push the final project to GitHub.

## Planned Agents
- Orchestrator Agent
- Content Writer Agent
- LaTeX Builder Agent
- QA Structure Agent
- QA BiDi Agent
- QA References Agent

## QA Plan
The QA process will check:
- Required Markdown files exist.
- LaTeX source files exist.
- CLS file exists.
- Bibliography file exists.
- Table of contents exists.
- PDF output exists.
- README includes setup and run instructions.
- Screenshots and logs are included.

## Previous Feedback Improvement
This assignment improves the weak points from the previous feedback:
- Better README documentation
- Clear setup instructions
- Screenshots as evidence
- QA and evaluation checklist
