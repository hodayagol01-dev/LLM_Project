# PLAN – Bonus Assignment Development Plan

## Phase 1 – Environment Setup

- Create a structured project folder.
- Create a Python virtual environment using venv.
- Install the required dependencies.
- Generate requirements.txt for dependency management.

## Phase 2 – Ollama Setup

- Install Ollama on the local machine.
- Pull a small local model for fast testing.
- Verify that the model responds through the terminal.

## Phase 3 – API Integration

- Create a configuration file.
- Define the local Ollama API URL.
- Define the selected local model.
- Implement a Python API client that sends direct HTTP requests to Ollama.
- Send the user prompt and conversation history to the API.

## Phase 4 – GUI Development

- Build a desktop chat window using DearPyGui.
- Add a chat display area.
- Add a user input field.
- Add a Send button.
- Display the model response inside the chat window.

## Phase 5 – Testing

- Verify that Ollama is installed and available.
- Verify that the selected local model is available.
- Run a direct API smoke test.
- Run the desktop GUI application.
- Confirm that the user can send a message and receive a response.

## Phase 6 – Documentation and Submission

- Write README documentation.
- Write PRD documentation.
- Write PLAN documentation.
- Write TODO documentation.
- Write API documentation.
- Write QA report.
- Push the final project to GitHub under the main course repository.
